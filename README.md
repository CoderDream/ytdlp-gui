# ytdlp-desktop

A [yt-dlp](https://github.com/yt-dlp/yt-dlp) desktop app

![image](https://github.com/user-attachments/assets/c66f46c9-862e-4202-ba8c-8420de08a765)


## Project Setup

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
