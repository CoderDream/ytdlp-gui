# ytdlp-desktop

A [yt-dlp](https://github.com/yt-dlp/yt-dlp) desktop app

![image](https://github.com/user-attachments/assets/cbefa75a-e650-4f3c-b215-782b06b1e0fd)



## Project Setup

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
